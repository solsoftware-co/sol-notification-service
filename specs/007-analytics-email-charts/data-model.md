# Data Model: Analytics Email Charts

**Feature**: 007-analytics-email-charts
**Date**: 2026-03-04

> No database schema changes are required for this feature. All chart data is derived from the existing `AnalyticsReport` payload (already fetched from GA4) and rendered at email-generation time.

---

## Existing Types Used (unchanged)

These types from `src/types/index.ts` are the data sources for the charts:

### `DailyMetric`
```ts
interface DailyMetric {
  date: string;       // ISO 8601 date, e.g. "2026-02-23" ‚Äî x-axis labels
  sessions: number;   // y-axis value for area chart
  activeUsers: number;
  newUsers: number;
}
```

### `TrafficSource`
```ts
interface TrafficSource {
  source: string;   // bar label, e.g. "google", "direct"
  sessions: number; // bar width value
}
```

### `TopPage`
```ts
interface TopPage {
  path: string;  // bar label, e.g. "/", "/about"
  views: number; // bar width value
}
```

## Modified Types

### `EmailAttachment` ‚Äî updated in `src/types/index.ts`

**Before (broken)**:
```ts
interface EmailAttachment {
  filename: string;
  content: Buffer | string;
  headers?: {
    'Content-ID': string;
    'Content-Disposition'?: string;
  };
}
```

**After (correct)**:
```ts
interface EmailAttachment {
  filename: string;
  content: Buffer | string;
  content_id?: string;    // CID for inline images, e.g. 'banner_image.png'
  content_type?: string;  // MIME type, e.g. 'image/png'
}
```

## New Internal Types

### `ChartBuffer` ‚Äî internal to `src/lib/charts.ts`

Not exported. Each chart generation function returns `Buffer` (PNG bytes) directly.

```ts
// Conceptual signature:
async function generateDailyTrendChart(metrics: DailyMetric[]): Promise<Buffer>
async function generateTopSourcesChart(sources: TrafficSource[]): Promise<Buffer>
async function generateTopPagesChart(pages: TopPage[]): Promise<Buffer>
```

## Updated Template Props

### `AnalyticsEmailProps` ‚Äî updated in `src/emails/templates/analytics-report-v1.tsx`

The `charts: Array<...>` prop is replaced with three optional per-section props:

**Before**:
```ts
type AnalyticsEmailProps = {
  // ...existing fields...
  charts: Array<{ title: string; description: string; image: string }>;
};
```

**After**:
```ts
type AnalyticsEmailProps = {
  // ...existing fields unchanged...
  dailyChart?: string;    // CID ref: 'cid:chart_daily'  ‚Äî absent = table-only
  sourcesChart?: string;  // CID ref: 'cid:chart_sources' ‚Äî absent = table-only
  pagesChart?: string;    // CID ref: 'cid:chart_pages'   ‚Äî absent = table-only
};
```

## Chart Dimensions

| Chart | Width | Height | Notes |
|-------|-------|--------|-------|
| Daily trend (area) | 760px | 220px | Fixed height; fits 7-day label row |
| Top sources (h-bar) | 760px | 40px √ó N + 40px | Dynamic; ~240px for 5 sources |
| Top pages (h-bar) | 760px | 40px √ó N + 40px | Dynamic; ~320px for 7 pages |

Charts are generated by the QuickChart API and returned as PNG at the requested dimensions.
